# Wireframes — locked design source of truth

The locked wireframe (Claude Design export, `StructTech Wireframes.dc.html`) covers all frames:

- **1a–1e** — first-pass structural screens (shell, pipeline, estimating, coordination, portal)
- **2a–2i** — agency layer, per-tenant stages, sign-off/material gating, expandable milestones, plus login/tenant-select, journey flow, and Delivery admin
- **3a** — Field crew module: Today home, sub-60s daily check-in, visual production packet (outdoor mode, role = crew)

**File:** `StructTech-Wireframes-LOCKED.html` (open in a browser; `support.js` renders it). This is the locked source of truth for the build. Targeted hi-fi is planned only for the client-facing surfaces: client portal (2e) and live estimating (2c).
