## LEVEL 1 — ON-SITE ESTIMATE SALES SYSTEM

**Why (problem statement):**
Leads call in and get qualified by feel. The roof scope, site survey, CompanyCam photos, and Pocket notes all happen — but the estimate happens LATER, back at the desk. Every day between survey and estimate is a day the homeowner shops competitors. The fix: qualify with a real checklist, estimate BEFORE the visit, validate on the roof, and hand them a number on the spot.

**Milestones:**
- [STRUCTTECH] Map the current lead-to-estimate flow, then build the custom qualification checklist + preliminary estimate engine — intake gathers the right data on the first call, scope order triggers at scheduling, and a preliminary estimate is ready before the truck rolls
- [CLIENT] Run every incoming lead through the qualification checklist — no gut-feel qualifying, no missing data fields
- [CLIENT] Roof scope ordered at appointment scheduling, preliminary estimate reviewed before every site visit
- [CLIENT] At the survey: validate, adjust, and present the estimate on-site — target under 2 minutes of modifications (CompanyCam + Pocket keep working exactly as they do today)
- [CLIENT] **WIN CONDITION:** Three straight jobs quoted on-site, same visit
- [STRUCTTECH] Train the process end-to-end and document it — intake to on-site number without Jacob in the loop

## LEVEL 2 — WORK ORDER, MATERIALS & SCHEDULING SYSTEM

**Why (problem statement):**
After the signature, the job lives in the project coordinator's head: detail meeting, work scope, work order, schedule, dumpster, material list, orders, delivery dates — all manual, all disconnected. One missed ready-by date stalls a crew. The fix: one flow where the signed estimate becomes the work scope, the work order builds the schedule, and materials track themselves to the job site.

**Milestones:**
- [STRUCTTECH] Build the post-sign workflow system — homeowner detail-meeting checklist (colors, finishes, expectations, zero surprises), work scope → work order generation, scheduling board, material list generated from the scope, order tracking with ready-by dates that update the work order automatically
- [CLIENT] Project coordinator runs the detail meeting off the checklist on every signed job — homeowner signs off on colors/finishes/details before anything is ordered
- [CLIENT] Every work order built from the system scope — dumpster and site necessities scheduled from the work order, not from memory
- [CLIENT] All material orders placed through the material list with ready-by dates logged — delivery dates visible on the schedule
- [CLIENT] **WIN CONDITION:** Three straight jobs from signature to material delivery with zero emergency supply runs and zero homeowner surprises
- [STRUCTTECH] Train the project coordinator as system owner and document the flow — signature to scheduled, materials inbound, hands off

---

## LEVEL 3 — VISUAL PRODUCTION MANAGEMENT SYSTEM

**Why (problem statement):**
The crew shows up with a work order, not a picture. Which trims, which boots, where the customs go — that knowledge lives in whoever sold the job. The fix: every job gets a visual production packet — trims, boots, placements, custom details mapped onto the scope and photos — so the production manager runs the job site without a single "call Isaac" moment.

**Milestones:**
- [STRUCTTECH] Build the production document generator — visual job packets assembled from the work scope, CompanyCam photos, and material list: trim map, boot/vent placements, custom detail callouts, everything the crew needs on one screen
- [CLIENT] Production manager creates the visual packet before every job start — no job kicks off without one
- [CLIENT] Crew works from the packet on-site — questions get answered by the document, gaps get logged to improve the next packet
- [CLIENT] **WIN CONDITION:** Three straight jobs run start-to-finish from the visual packet with zero clarification calls back to sales
- [STRUCTTECH] Train the production manager as packet owner and document the standard — sales sells, coordination schedules, production executes, nobody translates
